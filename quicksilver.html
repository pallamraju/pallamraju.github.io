QUICKSILVER - A Network Integration Platform

The world’s first application-aware and vendor-agnostic network automation solution, Mulesoft Quicksilver accelerates the journey to automation by connecting and choreographing your infrastructure and applications. At the intesection of software defined networking (SDN) and network function virtualization (NFV), Quicksilver brings the power of API-led connectivity to network infrastructure.

Overview

Quicksilver is an automation software that allows to self-regulate the configuration, management, testing, deployment and operations of physical and virtual devices within a network. This can be achieved only with Programmable Networks (device should be capable of sending or receiving network commands defined through software) and Intent-Based Networking.
